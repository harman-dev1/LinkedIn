# 🌐 LinkSphere (LinkedIn Clone)

A lightweight full-stack professional networking platform inspired by LinkedIn.  
Built using modern web technologies to demonstrate authentication, CRUD operations, and social networking features.

---

## 🚀 Live Demo
> *(yet to be added)*
https://your-deployed-link.com

---

## 📌 Features

- 👤 User authentication with **JWT-based secure login/signup**
- 🧑‍💼 User profile creation and management (CRUD operations)
- 📰 Create, edit, delete posts
- ❤️ Like and 💬 comment on posts
- 🤝 Send and manage connection requests
- 📡 Real-time-like activity feed (posts from network)
- 📱 Fully responsive UI for all devices
- ⚡ Secure backend APIs with Express & Node.js

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind CSS

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Authentication:**
- JSON Web Tokens (JWT)

**Deployment:**
- Render

---


---

## 🔐 Authentication Flow

- User registers / logs in
- Server validates credentials
- JWT token generated and stored
- Token used for protected routes (profiles, posts, connections)
